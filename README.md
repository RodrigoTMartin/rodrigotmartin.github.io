# rodrigotmartin.github.io

Personal academic website of Rodrigo Martin.

Built with [Academic Pages](https://github.com/academicpages/academicpages.github.io), a Jekyll theme for GitHub Pages.

## How to publish

1. Create a new repository on GitHub named exactly `RodrigoTMartin.github.io`
2. Push this folder to that repo:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/RodrigoTMartin/RodrigoTMartin.github.io.git
   git push -u origin main
   ```
3. Go to the repo Settings → Pages → set source to `main` branch
4. Your site will be live at https://rodrigotmartin.github.io in a few minutes

## Add your profile photo

Place a photo named `profile.jpg` in the `images/` folder.
